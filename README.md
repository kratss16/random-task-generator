## 📝 Описание проекта

«Генератор случайных задач» — это desktop-приложение с графическим интерфейсом, которое помогает пользователю получать случайные задачи из заранее созданного списка. Программа позволяет:

- ✅ Добавлять новые задачи с указанием типа (учёба, спорт, работа)
- 🔍 Фильтровать задачи по типу
- 🎲 Генерировать случайную задачу из отфильтрованного списка
- 📜 Просматривать историю последних 10 сгенерированных задач
- 💾 Автоматически сохранять все данные в JSON-файл
- 🗑️ Удалять ненужные задачи
